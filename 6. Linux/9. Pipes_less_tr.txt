Input and output
    - each command have input and output
    - we can use output of one command as input of another command  

Pipe character :
    - to give output of one command as input of another command

less : 
    - to display output of command in pagination
    - helps when printing large files.

    next page : space or f
    prev page : b
    scroll down : bottom arrow
    scroll up : upper arrow
    start of file : g
    end of file : G

Examples:
    cat /var/log/syslog | less
    ls /user/bin | less
    history | less

tr : translate or delete characters

ex. 
    1. echo "hello world" | tr 'a-z' 'A-Z'
        # output : HELLO WORLD

       echo "Hello World" | tr 'A-Z' 'a-z'
        # output : Hello World

       echo "hello" | tr 'el' 'ax'
        # Output: haxxo (e→a, l→x)

    2. cat file1.txt | tr 'a-z' 'A-Z'
        # all content will be capitalized

    3. cat file1.txt | tr 'a-z' 'A-Z' > newFile.txt
        # store capitalize content of file1 to newFile.txt

    4. deletion of characters
        echo "hello123world" | tr -d '0-9'
        # Output: helloworld

        echo "a,b,c,d" | tr -d ','
        # Output: abcd
