cat : concatenate

    1. cat fileName.txt
        - display content of file

    2. cat fileName.txt file2.txt file3.txt
        - display content of multiple file by concatenate it

    3. creating file 
        cat > newfile.txt
        type content and to save press ctrl + d to save ctrl + c to exit

    Options:
        cat -n filename.txt : display line number for all the files

    4. Combine multiple file into one
        cat file1.txt file2.txt > combined.txt

    5. Appending the contents
        cat additional.txt >> existing.txt
