head and tail:
    The head and tail commands are Unix/Linux utilities for viewing parts of files:

    head:
        head file.txt           # Shows first 10 lines
        head -n 5 file.txt      # Shows first 5 lines
        head -20 file.txt       # Shows first 20 lines
        head -c 100 file.txt    # Shows first 100 bytes

    tail:
        tail file.txt           # Shows last 10 lines
        tail -n 5 file.txt      # Shows last 5 lines
        tail -20 file.txt       # Shows last 20 lines
        tail -f logfile.txt     # Continuously shows new lines added to file

diff:
    compares files line by line and shows the differences between them

sort :
    organizing data in ascending or descending order based on various criteria

    sort file.txt     # Sort lines alphabetically(lexographically)
    sort file1.txt file2.txt            # Sort combined contents of both files
    sort *.txt                          # Sort all .txt files together
        
    sort with pipe : 
        ls | sort                           # Sort directory listing
        cat file.txt | sort                 # Sort file contents via pipe

    reverse order:  
        sort -r file.txt

    numeric:
        sort -n numbers.txt  
        sort -nr numbers.txt    # numeric and reverse order

hostname: 
    The hostname command shows or sets the system's hostname (computer name).

    hostname        # display current hostname
    hostname -i     # display Ip address
    sudo hostname newname   # set hostname temporary

domainname: 
    - The domainname command displays or sets the system's NIS 
      (Network Information Service) domain name.

    domainname      # display current domain name
    sudo domainname newdomain   # set domain name

id : displays user and group identification information for the current user or a 
     specified user.

    id  # shows user id-name, group-id name and all the groups user belongs to
    id -u   # shows the user id of current user
    id -u john # shows the user id of john
    id -g      # shows the primary group id
    id -G      # show all the groups id

    id -un      # Show username instead of UID
    id -gn      # Show group name instead of GID
    id -Gn      # Show all group names

    # read about more option available
