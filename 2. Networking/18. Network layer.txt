The Network Layer (Layer 3) in the OSI model is responsible for delivering data 
packets between devices across multiple networks.

Routing:
    Routing is the process of selecting the best path for a packet to travel from 
    source to destination across multiple networks.

    - There are graph like structure of router
    - routing finds best path from them using algorithm like dijsktra

    - Note:
        in routing not each personal router takes part.
        ex. my router not connected with neighbors router,
        but routing happen in between service providers

    Routing table:
        - A routing table is a data structure in a router that stores information 
          about possible paths to various network destinations.
        - each router have routing table

        It contains following:
            1. Network layer address of destination
                delivery is called hop to hop delivery
                next address called next hop

            2. Network layer address of person who are sending packets
            3. Data you want to sent

Forwarding:
    Forwarding is the actual action of moving a packet from one interface to 
    another based on the routing table.

    Forwarding table:
        A forwarding table is derived from the routing table, but it's optimized 
        for fast lookup.
